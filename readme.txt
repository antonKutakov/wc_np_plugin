This plugin add ukrainian shipping method "Nova Poshta" to Woocommerce.

== Upgrade Notice ==
* Add shipping method "Nova Poshta" to Woocommerce.

== Description ==

This plugin can calculate shipping cost with "Nova Poshta" rates.

== Installation ==

1. Get API Key of Nova Poshta in user panel on <a href="novaposhta.com.ua">novaposhta.com.ua</a>.
2. Insert your API key on plugin settings page for getting access to Nova Poshta API.
3. Synchronize adresses from Nova Poshta by pressing button "Синхронізувати Довідник Адрес НП" in plugin settings page.
4. Set Ukraine shipping zone in Woocommerce shipping settings.
5. Enable Nova Poshta shipping method on plugin settings page or woocommerce shipping settings page.

== Frequently Asked Questions ==
Q: Why "Nova Poshta" shipping method isn`t available?
A: Please set Ukraine shipping zone in Woocommerce shipping settings.
Q: Where i can set cargo type of product?
A: In shipping options of product.
